### 依赖项

**provider: ^6.1.2**：用于Flutter的状态管理

**font_awesome_flutter: ^10.8.0**：用于awesome图标的显示

**dio: ^5.8.0+1**：用于发送网络请求

**bot_toast: ^4.1.3**：用于Toast消息提示
